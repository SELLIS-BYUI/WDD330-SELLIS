Notes:

"Template literals can be thought of as super-powered strings as they behave in
the same way as normal string literals, but with the extra power of string interpolation.
For this reason, it is not uncommon to see backticks used to createallstrings in ES6 code" (JavaScript: Novice to Ninja, 2nd Edition).


Code examples:

const name = `Siri`; `Hello ${ name }!`; << 'Hello Siri!'! const age = 39; `I will be ${ age + 1 } next year`; << 'I will be 40 next year'
