There are global variables for the can be used to get access to entire program.
However there are better and worst ways/times to do this because it sometimes
raises a risk. A risk of causing errors in our programs. Global variables are
part of properties on the global object aka the window object.

"In general, you should refer to global variables without using the window object;
it’s less typing and your code will be more portable between environments. An exception is
if you need to check whether a global variable has been defined. For example, the
following code will throw a ReferenceError if x has not been defined" (JavaScript:
Novice to Ninja, 2nd Edition).
